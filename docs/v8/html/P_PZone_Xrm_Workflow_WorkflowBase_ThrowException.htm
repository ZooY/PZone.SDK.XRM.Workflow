Remarks
Если влаг установлен, то при возникновении ошибки будет вызвано исколючение и выполнение процесса будет прекращено. Если флаг не установлен, то исключение вызвано не будет, выходной параметр HasError будет иметь значение true, а в параметре ErrorMessage будeт сообщение об ошибке.
